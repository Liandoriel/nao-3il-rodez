Cette application vous permet de visualiser les différentes données du NAO.
Elle est décomposée en 6 modules.

  • Le module Comportement : Permet de lancer et stopper des activités sur le robot.
  • Le module Actionneur : Permet de voir les différents actionneurs du robot
  • Le module Audio : Permet de configurer les parametres audios du robot
  • Le module Caméra : Permet de voir ce que le robot voit (En cours developpement)
  • Le module Capteur : Permet de voir l'état des différents capteurs du robot
  • Le module Système : Permet de voir les données systèmes du robot

Attention, en faisant ceci, vous déconnecter tous les utilisateurs de l'application
